# **App Name**: MediaMaestro

## Core Features:

- Image Compression: Compress images (JPG, PNG, WebP) with Low/Medium/High presets.
- Background Removal: Remove image backgrounds using AI with smooth edges and transparent PNG output; the AI tool should determine if adjustments or additional tools are necessary to ensure high-quality results.
- Passport Photo Maker: Automatically detect faces, crop images, and set white/blue backgrounds for passport photos based on predefined country sizes.
- Image Resizing: Resize images by pixels or percentage, maintaining aspect ratio.
- Image Cropping: Crop images with free crop and preset aspect ratios (1:1, 4:5, 16:9).
- Image Format Conversion: Convert image formats (PNG, JPG, WebP) in bulk.
- Image Enhancement: Upscale images using AI to 1x or 2x resolution; the tool will determine which parameters or settings for the AI model are most relevant.
- PDF Merging: Merge multiple PDF files into a single PDF document.
- PDF Splitting: Split a PDF document into multiple files by extracting selected pages.
- PDF to Word Conversion: Convert PDF documents to editable Word (DOCX) format.
- Word to PDF Conversion: Convert Word documents to PDF format.
- File Format Conversion: Instant PNG â†” JPG conversion with bulk support.
- ZIP Tools: Compress and extract ZIP files.
- Cloud Storage: Saves the last 3 files to the cloud for free accounts, increasing limits based on the account tier.
- History Tracking: Maintain history of the user's activities, for later review. The number of items saved depends on the account's tier.
- Theme Mode Toggle: Switch between light and dark themes, auto switching with system settings.

- Primary color: A vibrant purple (#9D4EDD) for a modern and creative feel.
- Background color: A light purple (#F5EEFF) for a soft and clean background.
- Accent color: A saturated blue (#560A86) for interactive elements and call-to-action buttons.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable design.
- Use minimalist, line-based icons for a clean and modern look.
- AdSense ad placements include a top banner, below output, sidebar (desktop), and footer, with placeholders defined for easy ad integration.
